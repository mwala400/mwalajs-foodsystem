# MwalaJS Food System

This project is built using **MwalaJS framework**.

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/USERNAME/mwalajs-foodsystem.git
cd mwalajs-foodsystem

Install dependencies:

npm install

Create .env file (example):

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=mwalajs_foodsystem
PORT=3000

Start the server:

mwala serve

Visit: http://localhost:3000
